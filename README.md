## Instalasi
Silakan untuk menginstall dependencies dengan mengetikan perintah ``npm install`` atau ``yarn install``. Tergantung dari package manager yang digunakan.

Untuk Testing :
```bash
yarn start-dev
```

Untuk Build :
```bash
yarn build
```
